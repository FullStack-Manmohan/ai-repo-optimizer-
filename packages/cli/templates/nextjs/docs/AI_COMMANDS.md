# Next.js Executive Commands

Useful system scripts and commands default to Next.js template environments.

---

## 🚀 Development Tasks
- **Start Next.js Local Server**: `npm run dev` (Runs server on [http://localhost:3000](http://localhost:3000))

## 🏗️ Build & Compiles
- **Compile / Packaging**: `npm run build`
- **Output Folder**: Out folder defaults to `.next/` or `out/` for static exports.

## 🩺 Code Health & Auditing
- **Lint Next.js**: `npm run lint` (uses next-lint)
- **Formatting Standards**: `npx prettier --write .` (if Prettier is active)
- **TypeScript Compiler Checks**: `npx tsc --noEmit`
